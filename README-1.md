# SaaS Subscription Metrics Analytics

## Files
- `saas_subscriptions.csv` - Sample SaaS subscription dataset
- `analysis.py` - Python script for KPI analysis
- `README.md` - Project documentation

## KPIs
- Total Customers
- Active Customers
- Cancelled Customers
- Monthly Recurring Revenue (MRR)
- Average Revenue Per User (ARPU)
- Churn Rate

## Run

```bash
pip install pandas
python analysis.py
```
